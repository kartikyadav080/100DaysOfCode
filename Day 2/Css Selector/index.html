Css Stands for Cascading style sheet

There are three types of Css

  • Inline
  • Internal
  • External

Today we will use Internal Css

Css Specificity

It defines specificity of a selector i.e which css will be apply on the element.
It is as follows:-

  1. Inline Styles
  2. Id Selectors
  3. Class selector
  4. Element Selector
  5. Universal Selector
